# 🎨 Personal Portfolio Website UI/UX Design

A modern and responsive portfolio website designed in Figma as part of the **Syntecxhub UI/UX Internship – Week 1**.

## 📌 Overview

This project showcases a clean and professional personal portfolio interface. The design focuses on usability, visual hierarchy, and responsiveness across multiple devices.

## ✨ Features

- Modern and minimal UI
- Responsive layouts (Desktop, Tablet & Mobile)
- Hero section
- About section
- Skills section
- Projects showcase
- Contact section
- Reusable UI Components
- Auto Layout
- Interactive prototype

## 🛠️ Tools Used

- Figma
- Auto Layout
- Components
- Prototype

## 📱 Responsive Designs

- Desktop
- Tablet
- Mobile
